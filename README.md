# ASC - Agentic Simulink CLI

LLM 辅助汽车嵌入式建模工具，基于 Claude/GPT 等大模型，提供 Simulink 模型的智能建模、分析与调试能力。

## 安装

从 [Releases](https://github.com/divinerIU/agentic-simulink-cli/releases) 页面下载最新安装包，双击运行即可。

安装完成后在命令行执行：

```bash
asc setup
```

## 功能特性

- 🔧 **Simulink 模型建模** — 通过自然语言描述自动生成 Simulink 模型
- 📊 **模型分析** — 智能分析模型结构、信号流和参数配置
- 🛠 **MATLAB 集成** — 无缝连接 MATLAB 工作空间，支持实时调试
- 🤖 **多 LLM 支持** — 支持 Anthropic Claude、OpenAI GPT 等多种模型
- 📝 **插件系统** — 可扩展的插件架构，支持自定义工具和技能

## 系统要求

- Windows 10/11 (x64)
- MATLAB R2024b+（可选，用于 MATLAB 集成）

## 许可证

仅供内部使用。
