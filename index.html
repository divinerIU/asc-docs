Agentic Simulink CLI

用自然语言操作 Simulink 模型的智能工程助手。你说需求，它帮你完成——读文件、改模型、跑命令、查代码，全程安全可控。

Get Started Developer Guide

核心能力

智能对话

用自然语言描述需求，ASC 理解你的意图并自动执行。支持复杂任务的多步推理。

安全操作

每次操作都经过多层安全检查。危险操作必须确认，所有操作都有审计日志。

项目记忆

ASC 记住你的项目信息、技术约定和工作进度。下次对话不用从头说起。

技能扩展

用 Markdown 定义工作流程，ASC 按你的 SOP 执行。团队共享，新人即用。

服务与集成

通过标准协议连接 MATLAB、Simulink 等外部工具。即插即用，自动重连。

透明界面

实时看到 ASC 在想什么、做什么。流式输出，确认对话框，完全可控。

快速开始

# 安装并启动
pip install -e .
asc setup

    # 在 ASC 会话内安装 Simulink 领域插件
    /plugins install simulink-agentic-toolkit

启动后，在终端里用自然语言和 ASC 对话即可。试试输入："帮我看看当前目录下有哪些 Simulink 模型"
